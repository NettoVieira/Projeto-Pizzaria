# pim

#### Compilar usando GCC

gcc -o pim pim.c

#### Executando
./pim

#### Arquivos

######  Pizzas
Padrão: id: nome registrado

ex:
```
1: quatro queijos
2: calabresa
```

######  Estoque
Padrão: id da pizza: quantidade em estoque - Nome do Produto

ex:
```
1: 50.000000 - Massa
2: 10.000000 - Orégano
```

######  Receita
Padrão: id da pizza: id do produto - quantidade do produto

ex:
```
1: 1 - 0.100000, 3 - 0.300000  
2: 5 - 0.500000, 0 - 0.300000  
```
